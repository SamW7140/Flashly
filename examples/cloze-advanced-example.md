## Cloze Deletion Best Practices

### ✅ Good Cloze Cards
- One concept per deletion
- Context helps recall
- Not too obvious, not too obscure
- Creates active learning

### ❌ Avoid
- Too many deletions (makes it confusing)
- No context (can't be answered)
- Too obvious (wastes time)
- Ambiguous answers

### Strategy
1. Read sentence fully first
2. Delete key terms
3. Ensure remaining text provides context
4. Test that each deletion creates a valuable question
